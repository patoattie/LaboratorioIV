ANGULAR
La meta es crear nuestros propios tags.
En app-root va la página web completa que generamos, es un gran AJAX que genera un spa (single page application).
Angular facilita el proceso de utilizar un componente.
Dentro de src vamos a generar una carpeta por cada componente que vayamos a generar.
src/assets -> Recursos (fotos, videos, docs)
src/environments -> Seteos que afectan a toda la app (ej. conexion a firebase o php)

ng generate (g) -> genera un nuevo componente.
c -> componente + ruta
class -> clase + ruta
